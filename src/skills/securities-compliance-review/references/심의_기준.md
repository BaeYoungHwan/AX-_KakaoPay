# 증권 투자권유·광고 심의 기준

이 문서는 Retrieval Agent의 법령 검색 범위, Analysis Agent의 위험도 판정
기준을 정의한다. 코드(`content_classifier.py`, `law_search.py`)를 고치지
않고 이 문서를 수정하면 심의 기준을 조정할 수 있다.

## 1. 콘텐츠 타입 분류
| 콘텐츠 타입 | 예시 |
| --- | --- |
| 투자권유문자 | MTS/HTS 푸시, 문자 안내 |
| 상품소개서 | 펀드, ELS/ELB, 해외주식, 채권 상품 안내 자료 |
| SNS광고 | 인스타그램/유튜브/블로그 등 디지털 광고 |
| 리서치·투자자문 코멘트 | 애널리스트 코멘트, 투자자문 메시지 |
| 신규상품 안내 이메일 | 이메일 뉴스레터, 신규 서비스 안내 |

## 2. 우선 검색 법령 (Retrieval Agent 우선순위)
1. 자본시장법 제49조 (부당권유행위 금지)
2. 자본시장법 제57조 (투자광고)
3. 금융투자업규정 중 광고 심의기준 관련 조항
4. 금융소비자보호법 부당권유 금지 조항

우선순위 법령에서 관련 조문이 검색되지 않을 때만 법제처 Open API 전체 검색
으로 범위를 넓힌다. 이렇게 하면 범용 검색보다 증권업 맥락에 맞는 조문이
먼저 잡힌다.

## 3. 키워드 확장 규칙
### 기존 (JB-Compliance-Agent 원본, 유지)
수익, 보장, 확정, 원금, 손실, 투자, 펀드, 개인정보, 문자, SMS, 푸시

### 증권 광고 특화 추가
- "손실 없이", "무조건", "확실히"
- "원금 보장", "확정 수익", "확정 금리"
- "국내 유일", "업계 최초", "타사 대비"
- "과거 수익률" (미래 수익 보장으로 오인될 수 있는 과거 실적 강조 표현)
- "지금 아니면", "한정", "선착순" (긴급성으로 합리적 판단을 저해하는 표현)

## 4. 위험도 판정 기준
| 조건 | 기본 위험도 | 근거 |
| --- | --- | --- |
| "원금 보장", "확정 수익", "확정 금리" 계열 표현 포함 | High | 투자상품을 예금처럼 오인하게 하는 표현은 자본시장법상 부당권유 소지가 매우 큼. LLM 판단 전에 규칙으로 우선 High를 부여하고, Analysis Agent가 조문 근거로 재확인 |
| "손실 없이", "무조건", "국내 유일", "업계 최초" 등 단정·과장 표현 포함 | Medium (조문 확인 후 상향 가능) | 과장 광고 소지 |
| "과거 수익률" 언급 + 향후 수익 예측 표현 동반 | Medium 이상 | 과거 실적의 미래 보장 오인 소지 |
| 긴급성 표현("지금 아니면", "선착순")만 단독 포함 | Low~Medium | 즉시 위반은 아니나 합리적 판단 저해 요소로 언급 |
| 위 키워드 없음 | Analysis Agent가 조문 근거로 판단 | 규칙으로 선판정하지 않고 LLM+근거 조문 판단에 맡김 |

규칙 기반 선판정은 LLM이 "원금보장" 같은 명백한 위반 표현을 과소평가하는
것을 막기 위한 안전장치이며, 최종 판단과 조문 인용은 Analysis Agent가
검색된 법령 조문을 근거로 다시 확인한다.

## 5. 검증 원칙 (Verification Agent, 원본 구조 유지)
- Analysis Agent 출력의 "위반 가능 조항"에 등장한 법령명이 Retrieval Agent가
  실제로 검색해 반환한 목록에 없으면 재분석을 요청한다.
- 재분석 후에도 근거 없는 법령이 계속 등장하면 "검증 실패, 준법 담당자 직접
  확인 필요"로 표시하고 자동 승인 후보에서 제외한다.

## 6. 피드백 저장 원칙 (Feedback Agent, 원본 구조 유지)
- 승인 건은 그대로 저장한다.
- 거절/수정 건은 피드백 메모 길이와 위반 법령 명시 여부를 확인해, 사유 없는
  거절/수정은 저장을 차단한다 (저품질 학습 데이터 방지).

## 7. 데이터 처리 원칙
- 상품소개서에는 미공개 신상품 정보가 포함될 수 있으므로, 원본과 동일하게
  로컬 LLM(Ollama EXAONE 3.5)과 로컬 ChromaDB를 사용하고 외부 API 호출은
  법제처 Open API 조문 검색으로 제한한다.
